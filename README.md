# Flask+React
## Initialization

### Flask 

```shell
# ROOT_DIR

# Create Python environment
> python -m venv .venv

# Install packages
> python install -r requirement.txt
```

### React 

```shell
# ROOT_DIR

> cd frontend
> yarn install
```

## Run

Click `run.bat`
